# Secure File Storage Backend

This is the backend for Secure File Storage System using Laravel 12 + Sanctum.

## Setup

1. Copy `.env.example` to `.env` and set your database credentials.
2. Install dependencies:
   ```bash
   composer install
   ```

Generate app key:

php artisan key:generate

Run migrations:

php artisan migrate

Serve the application:

php artisan serve
